## Prompt Detalhado para Geração de Imagem (nanobanana `generate_image`)

Este prompt foi cuidadosamente elaborado seguindo as diretrizes e princípios da extensão `nanobanana` para gerar uma imagem de cabeçalho de alta qualidade para o README do seu pacote Dart.

```
count: 1
prompt: "Header image for a GitHub README of a Dart package named 'OpenCNPJ Dart Client'. The package is used for querying, validating, and formatting Brazilian company data (CNPJ). The image should be rectangular, modern, clean, and incorporate elements representing: Dart/Flutter (e.g., Dart logo, subtle Flutter-like UI elements), Brazil (e.g., subtle Brazilian flag colors, abstract representation of Brazil map), Data/Information (e.g., abstract data visualization, database icons, code snippets), Companies/Business (e.g., abstract building shapes, gears, network nodes). Colors should be professional, vibrant, and reflect technology and Brazil. Avoid prominent text within the image itself. Style: Digital art, clean vectors, high detail, studio lighting. Aspect Ratio: 3:1"
```

**Explicação dos Elementos do Prompt e Adesão às Diretrizes do `nanobanana`:**

*   **`count: 1`**: Garante a geração de uma única imagem, aderindo ao "Precise Count Adherence".
*   **`prompt: "..."`**: A descrição detalhada inclui:
    *   **Estilo e Variação (`--styles`, `--variations` conceituais):** `rectangular, modern, clean, Digital art, clean vectors, high detail, studio lighting` definem o estilo artístico e visual.
    *   **Elementos Visuais:** `Dart/Flutter (e.g., Dart logo, subtle Flutter-like UI elements), Brazil (e.g., subtle Brazilian flag colors, abstract representation of Brazil map), Data/Information (e.g., abstract data visualization, database icons, code snippets), Companies/Business (e.g., abstract building shapes, gears, network nodes)` especificam os componentes visuais para consistência e relevância temática.
    *   **Cores:** `professional, vibrant, and reflect technology and Brazil` orienta a paleta de cores.
    *   **Texto:** `Avoid prominent text within the image itself` é crucial para a "Text Accuracy and Quality", permitindo que o texto seja adicionado posteriormente de forma controlada.
    *   **Requisitos Técnicos:** `rectangular` e `Aspect Ratio: 3:1` garantem o formato adequado para um cabeçalho de README.
*   **Qualidade e Padrões Criativos:** A combinação dos elementos visa uma imagem visualmente atraente e funcional, que atenda ao propósito de ser um cabeçalho profissional para um repositório GitHub.
